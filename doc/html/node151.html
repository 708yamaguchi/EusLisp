How to Start

The MARS program resides in robot/MARS/ver.XXX. First, copy the ".eursrc" file to your home directory. Make necessary changes. (Pathname of the directory in which MARS is installed.)

Invoke eusx in this directory, and all the files are automatically loaded and the simulator starts running. Wait until a window opens and an initial message appears in the bottom window.

Example: 

Try "SYSTEM"->"Load" menu.
Load "example.bbs".
And "SCL"->"On" menu.

Notes:
SCL "Off" Pauses the simulation but continue GUI processing.
SYSTEM "Quit" Exits the toplevel loop. Can be resumed by (mars-loop).
SYSTEM "Save" Saves the current state to a file.
SYSTEM "All-Clear" Clears everything and initializes the system.
SYSTEM "Reset" Is used to reset internal states of the robots (Special purpose).
